如何快速获取ssh Key密钥

正文

  1. 在git bash执行一下指令

git config --global user.email "you@example.com"
git config --global user.name "Your Name"

  1. 生成shh key

    ssh-keygen -t rsa

  2. 一路回车就行

  3. 默认会生成在目录

    C:\Users\用户名\.ssh

  4. 获取生成目录下id_rsa.pub里的密钥就能使用了
